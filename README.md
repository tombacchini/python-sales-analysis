# 📊 Analyse des ventes avec Python

## 🎯 Objectif du projet

Ce projet a pour objectif d’analyser la performance commerciale d’une entreprise à partir de données de ventes.

L’analyse a été réalisée avec Python afin de répondre à différentes problématiques business : chiffre d’affaires, rentabilité, performance produit et évolution dans le temps.

Ce projet s’inscrit dans une démarche globale d’analyse de données, en complément d’un projet réalisé en SQL sur le même dataset.

## 🛠️ Outils utilisés

- Python
- Pandas
- Matplotlib
- Google Colab

## 📁 Dataset

Le dataset utilisé contient des informations sur :
- les ventes (net_revenue)
- les produits (product_name)
- les clients (client_name)
- les régions (region)
- les dates (sale_date)
- les marges (margin_pct)

Il s’agit d’un dataset fictif utilisé dans un cadre d’apprentissage.

Ce dataset est identique à celui utilisé dans le projet SQL, ce qui permet de comparer les analyses réalisées avec deux outils différents.

## 📊 Analyses réalisées

### 1. Performance par catégorie
- Identification des catégories générant le plus de chiffre d’affaires

### 2. Analyse de la rentabilité produit
- Identification des produits avec les meilleures marges moyennes

### 3. Analyse géographique
- Calcul de la contribution de chaque région au chiffre d’affaires total
- Visualisation de la répartition des revenus

### 4. Analyse temporelle
- Évolution mensuelle du chiffre d’affaires
- Identification des tendances dans le temps

### 5. Relation chiffre d’affaires / rentabilité
- Analyse de la corrélation entre volume de ventes et marge
- Visualisation via un scatter plot

### 6. Analyse client
- Calcul du panier moyen par client

### 7. Analyse statistique
- Étude de la distribution du chiffre d’affaires
- Détection de la dispersion des données

## 🔍 Principaux insights

- Certaines catégories dominent le chiffre d’affaires
- Les produits les plus rentables ne sont pas toujours les plus vendus
- Les revenus sont répartis entre plusieurs régions, sans dépendance unique
- Il n’existe pas de corrélation significative entre chiffre d’affaires et marge
- La distribution des ventes montre une variabilité importante

## 🚀 Compétences mises en œuvre

- Manipulation de données avec Pandas
- Agrégation et groupby
- Analyse temporelle avec les dates
- Visualisation de données avec Matplotlib
- Analyse statistique
- Interprétation business des résultats
